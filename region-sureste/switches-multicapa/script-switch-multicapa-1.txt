enable
configure terminal
hostname MLS1
ip domain-name juanmark.com
no ip domain-lookup
service password-encryption
enable secret Cisco123!
username admin secret Admin123!


crypto key generate rsa general-keys modulus 2048
line vty 0 15
login local
transport input ssh
exit
line console 0
login local
exit

ip routing

vtp mode client
vtp domain JUANMARK
vtp password Cisco123!
