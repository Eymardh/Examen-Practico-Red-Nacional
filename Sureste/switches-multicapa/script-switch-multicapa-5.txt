Multicapa Switch 5 -VTP Client

 
enable
configure terminal
hostname MLS5
ip domain-name juanmark.com
no ip domain-lookup

username admin secret Admin123!

crypto key generate rsa general-keys modulus 2048
line vty 0 15
login local
transport input ssh
exit

ip routing

vtp mode client
vtp domain JUANMARK
vtp password Cisco123!

---- Configurar STP
spanning-tree mode pvst
spanning-tree vlan 65 priority 8192
spanning-tree vlan 66 priority 8192
spanning-tree vlan 67 priority 8192
spanning-tree vlan 68 priority 8192
spanning-tree vlan 99 priority 8192

--- Configuración EtherChannel 3: MLS5 - MLS0
interface range GigabitEthernet1/0/6-7
description EtherChannel 3 to MLS0
switchport mode trunk
switchport trunk allowed vlan 65,66,67,68,99
switchport nonegotiate
channel-group 3 mode desirable
no shutdown
exit
